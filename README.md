# Adapty.io Clone

This project is a clone of the [Adapty.io](https://adapty.io) website, built using [Next.js](https://nextjs.org). The goal is to replicate the design and functionality of the original website as closely as possible.

## Prompts Used to Create This Project

1. **Initial Prompt**: 
   - "Clone the website exactly like this website: https://adapty.io/"
   - "Fix the hero section to match the original design."

2. **Responsive Design Prompt**:
   - "Ensure the hero section fits 100% into the screen and is device-friendly."

3. **Typography and Layout Prompt**:
   - "Fix the typography and make the hero section responsive for all devices."

4. **Overflow Issue Prompt**:
   - "The hero section is overflowing; fix it to fit the viewport properly."

5. **README Update Prompt**:
   - "Update the README and include the prompts used to create this project and other details."

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

## Features

- **Framework**: Built with Next.js for fast and efficient development.
- **Styling**: Tailwind CSS for responsive and modern design.
- **Animations**: Framer Motion for smooth animations.
- **Font Optimization**: Uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to load the Geist font family.
- **Responsive Design**: Fully responsive and optimized for all devices.
- **Hero Section**: Matches the original Adapty.io hero section design, including typography, layout, and responsiveness.
